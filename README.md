# UK Pathways Hub prototype

This folder contains a simple static website designed for GitHub Pages.

## Files
- `index.html` – home page
- `live.html`, `work.html`, `learn.html`, `employers.html`, `contact.html`
- `before-you-arrive.html`, `after-you-arrive.html`, `living-in-london.html`
- `working-in-uk.html`, `sectors.html`, `tool.html`, `disclaimer.html`
- `style.css` – shared design
- `script.js` – simple logic for the London Pathway Tool

## How to publish on GitHub Pages
1. Create a free GitHub account.
2. Create a new public repository.
3. Upload all files from this folder.
4. Go to **Settings** → **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Choose the **main** branch and **/(root)** folder.
7. Save.
8. Wait a minute or two. GitHub will show the live website link.

## How to edit later
- Open any HTML file on GitHub.
- Click the pencil icon.
- Change the text.
- Commit the change.
- GitHub Pages republishes the site automatically.
