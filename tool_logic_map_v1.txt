Tool logic map v1

The updated tool combines:
- sector profile
- country profile
- years of experience
- type of experience
- English confidence for work
- sponsorship need
- regulated role question
- current location
- main goal

Hidden dimensions:
- skill translation
- experience strength
- communication readiness
- recognition need
- entry readiness
- sponsorship fit
- London fit
- career continuity
- barrier level

The tool turns those hidden dimensions into six visible result areas:
- Your likely route
- What may help most
- Main challenges
- London view
- Best next steps
- Country note if country was not matched
